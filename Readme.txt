Key Bindings:

Key : Button
"0" : 0, "1" : 1, "2" : 2, [...] "9" : 9
"*" = "*", "/" : "/", "+" : "+", "-" : "-"
"." = ".", "(" : "(", ")" : ")"
"r" : "SqrRoot", "#" : "**"
